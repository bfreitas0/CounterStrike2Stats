This is a Next.js project where you can look up your stats for the video game CS2 using your steam id.

To run, you must create a .env file with your database credentials and your steam api key.

Apologies for the single commit...

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
